# Project Alpha – Frontend Client

This repository contains the frontend client for Project Alpha, an educational quiz platform designed to improve student engagement and knowledge retention in non-STEM subjects through interactive multiple-choice quizzes.

The frontend provides the user interface for both teachers and students, enabling quiz creation, participation, and feedback.

## Overview

The frontend is built using HTML, CSS, and JavaScript, focusing on simplicity, accessibility, and engagement. The UI is designed to support short, focused learning interactions and minimise cognitive load for students.

---

## Technologies Used

- HTML – Page structure and content
- CSS – Styling and responsive layout
- JavaScript – Client-side logic and API communication
- Fetch API – Communication with backend services

---

## UX Design

User experience design focused on clarity, ease of use, and engagement for both teachers and students. The interface prioritises minimal navigation, clear call-to-action buttons, and immediate feedback during quizzes.

## Getting Started
### Prerequisites

Ensure you have the following installed:

- A modern web browser (Chrome, Firefox, Edge)
- Git

---

### Installation

1. Clone the repository:
```
git clone git@github.com:Maisie90/alpha-project.git
```

2. Navigate to the frontend directory:
```
cd project-alpha/frontend
```

### Running the application
Open index.hmtl directly into your broswer.
